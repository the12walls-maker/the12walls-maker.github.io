Markets, Events & Printables

Events Calendar

The Events Calendar shows all your upcoming and past market events in a calendar view. It also lets you set target quantities for each event so your Prep List knows what you plan to bring.

Getting there

Tap Events on the home screen.

Adding an event

  1. Tap the + button or tap an empty date on the calendar.
  2. Select the Market this event belongs to.
  3. Set the date (and optionally the start/end time).
  4. Toggle Recurring if this event repeats (weekly, bi-weekly, monthly).
  5. Tap Save.

Setting event targets

Event targets are the quantities you plan to bring to a specific market. They feed into the Prep List.

  1. Tap any event on the calendar.
  2. Tap Set Targets.
  3. For each recipe/product, enter the number of units you plan to bring.
  4. Tap Save.

The Prep List combines these targets with active customer orders to calculate how much you need to bake.

Recurring events

When you create a recurring event:

  • The app generates occurrences based on the recurrence rule.
  • You can edit a single occurrence or all future occurrences.
  • Deleting a single occurrence does not affect the rest of the series.

Calendar views

Toggle between Month, Week, and Agenda views using the tabs at the top of the calendar. The Agenda view shows a list of upcoming events sorted by date.

Market day report

Tap any event and choose Print Market Day Report to generate a PDF summary of the event's sales history, orders, and any notes. This requires the Printables feature.
