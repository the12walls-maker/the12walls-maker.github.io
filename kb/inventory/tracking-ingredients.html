Inventory

Tracking Ingredients

The Inventory page lets you record what you have on hand, set reorder thresholds, track supplier prices, and log waste.

Getting there

Tap Kitchen on the home screen, then tap Inventory — or tap the Inventory card directly from the home screen if shown.

Adding an ingredient

  1. Tap Add Ingredient (or whatever your craft calls it).
  2. Enter:
  • Ingredient name (e.g., "Bread Flour")
  • Current quantity — how much you have right now
  • Unit (kg, lb, oz, cups, each, etc.)
  • Reorder threshold — the quantity at which you want a low-stock alert
  • Cost per unit — what you paid per unit (used in COGS calculations)
  • Supplier — optional, for your reference
  1. Tap Save.

Editing an ingredient

Tap the edit icon on any row to change quantities, costs, or the reorder threshold.

Updating the reorder threshold

Tap the reorder threshold value on any row to quickly edit it inline without opening the full editor.

Supplier price history

When you update the cost per unit for an ingredient, StallVantage records the previous price. An arrow-up icon on the cost column indicates the price has changed since it was last set. Hover over (or tap) the icon to see what the previous cost was.

Finished goods on hand

The Finished Goods On Hand section lets you record how many finished units (e.g., loaves, jars, bags) you already have ready to sell. This feeds into the Prep List so it does not tell you to bake what you already made.

  1. Find the recipe in the table.
  2. Tap the + icon (or edit icon if a count exists) to set the current on-hand count.
  3. Enter 0 to remove the entry.

Units display

StallVantage stores weights internally and can display them in either pounds or kilograms based on your preference. Change this in Settings → Preferences → Weight unit.
