Kitchen & Production

Baking Tools

The Baking Tools page contains quick-access calculators for common baking math: unit conversion and batch scaling.

Getting there

Tap Kitchen on the home screen, then tap Baking Tools.

Unit Converter

Convert between common baking units instantly:

  • Weight — grams, ounces, pounds, kilograms
  • Volume — cups, tablespoons, teaspoons, milliliters, liters, fluid ounces
  • Temperature — Fahrenheit and Celsius

Enter a value in any field and the equivalent amounts in all other units update in real time.

Batch Scaler

Scale any recipe up or down without manual math:

  1. Enter the original batch size (e.g., 12 cookies).
  2. Enter the target batch size (e.g., 36 cookies).
  3. The scaler calculates the multiplier and shows the adjusted amount for each ingredient.

You can type in ingredient amounts directly or link to a saved recipe. Scaled values can be copied to the clipboard for use in notes or shared with your team.

Tips

  • Use the Batch Scaler before updating a recipe — it shows the new ingredient quantities before you commit to changing your recipe card.
  • The temperature converter is handy when following recipes from other countries that use Celsius if your oven displays Fahrenheit, or vice versa.
